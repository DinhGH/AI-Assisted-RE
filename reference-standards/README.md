# Reference Standards

Thư mục này dùng để lưu **tài liệu chuẩn** phục vụ đánh giá requirement.

## Bạn có thể bỏ gì vào đây?

- PDF tiêu chuẩn nội bộ/quy chuẩn nghiệp vụ
- Guideline viết requirement
- Checklist QA/BA
- Tài liệu domain đặc thù

## PDF có dùng được không?

**Có.** Bạn có thể thả trực tiếp file `.pdf` vào thư mục này.

## Lưu ý hiện tại

- Hệ thống đã có nơi chứa tài liệu chuẩn.
- Engine đã được tích hợp để tự động đọc tài liệu trong thư mục này (bao gồm PDF) khi phân tích requirement.
- Các tài liệu chuẩn được dùng để:
  1. tạo tập mẫu tham chiếu cho similarity,
  2. tăng độ thực tế khi chấm completeness (measurable/condition/domain alignment),
  3. mở rộng nhận diện ambiguity với từ mơ hồ phổ biến.
- Khi bạn thêm/sửa file chuẩn, hãy restart `ai-engine` để nạp lại nội dung chuẩn mới nhất.
