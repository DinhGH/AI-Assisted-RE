# Frontend

<!--
  This folder contains the React/Vite frontend for uploading requirements,
  browsing extracted items, viewing quality metrics, and chatting with the AI helper.
-->

## Purpose

Host the user interface for requirement upload, analysis dashboards, AI chat, and requirement viewing.

## Responsibilities

- Present input forms and document upload flows
- Visualize quality scoring and trend data
- Provide a chat surface for requirement refinement
- Keep UI logic separate from backend and AI concerns

## Example usage

Run `npm install` and `npm run dev` to launch the UI locally.
