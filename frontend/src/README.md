# frontend/src

<!--
  This folder is the main React source root.
  It exists to separate application code from configuration and build tooling.
-->

## Purpose

Hold the UI application code for the Vite-based frontend.

## Responsibilities

- Define the app shell and routing composition
- Organize reusable UI features and cross-cutting client code
- Keep feature modules isolated and easy to scale

## Example usage

Start new work from feature folders under `frontend/src/features/` and shared UI under `frontend/src/components/`.
