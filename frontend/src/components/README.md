# components

<!--
  This folder contains reusable, presentation-focused UI building blocks.
  It exists to prevent feature pages from duplicating visual primitives.
-->

## Purpose

Store shared React components used across multiple pages and features.

## Responsibilities

- Provide reusable UI primitives
- Keep visual logic separate from feature behavior
- Support consistent styling and accessibility patterns

## Example usage

Put buttons, cards, charts wrappers, dialogs, and other shared widgets here.
